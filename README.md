# CRUD Laravel de Contatos

## Funcionalidades implementadas:

- Listagem de contatos
- Criação de novo contato com validação de nome, contato e email
- Edição de contatos
- Visualização de detalhes do contato
- Exclusão lógica (soft delete) com lixeira e restauração
- Mensagens de sucesso e erro
- Layout com Bootstrap
- Testes unitários e feature testando validação
